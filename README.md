# CarbonAI Enterprise

AI system for real-time enterprise cost intelligence and autonomous carbon capture optimization.

## Problem
Carbon capture systems often operate with static parameters, leading to energy waste and cost inefficiencies.

## Solution
CarbonAI predicts CO₂ spikes using AI and recommends autonomous optimization actions while calculating cost savings.

## Features
- AI-based CO₂ prediction
- Simulated industrial sensors
- Optimization engine
- Cost savings estimation
- Live React dashboard

## Tech Stack
Backend: FastAPI, Scikit-learn
Frontend: React
